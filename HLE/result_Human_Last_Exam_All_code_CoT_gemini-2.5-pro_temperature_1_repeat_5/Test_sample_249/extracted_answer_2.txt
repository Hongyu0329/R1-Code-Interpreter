Let $D$ be the minimum diameter. The value is given by the piecewise formula:
$D(n,m) = \begin{cases} 2 & \text{if } n-m+2 = 1 \text{ (i.e., } m = n+1) \\ 3 & \text{if } n-m+2 = 2 \text{ (i.e., } m = n \text{, requires } n \ge 2) \\ 4 & \text{if } n-m+2 \ge 3 \text{ and } 2m \ge n+1 \\ 2 \lceil \frac{n-m}{m} \rceil + 2 & \text{if } n-m+2 \ge 3 \text{ and } 2m < n+1 \end{cases}$
