The expression for the magnetic field vector `H` at a position `(x, y, z)` for an applied field `H_a > H_0` and in the region `|x| >> w` is:

`H_x(x, z) ≈ -sgn(x) * ( (2π J_c d w²) / D² ) * sin(2πz / D) * exp(-2π|x| / D)`
`H_y(x, z) = 0`
`H_z(x, z) ≈ H_a + ( (2π J_c d w²) / D² ) * cos(2πz / D) * exp(-2π|x| / D)`
