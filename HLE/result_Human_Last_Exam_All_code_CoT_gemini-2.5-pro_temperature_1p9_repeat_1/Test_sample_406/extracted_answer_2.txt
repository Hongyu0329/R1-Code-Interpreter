Assuming (X,d_X), (Y,d_Y), and (Z,d_Z) are complete metric spaces and F, G, H satisfy the contraction inequalities `d_X(F(..),F(..)) ≤ k₁d_X+k₂d_Y+k₃d_Z`, `d_Y(G(..),G(..)) ≤ l₁d_Y+l₂d_X+l₃d_Y`, and `d_Z(H(..),H(..)) ≤ m₁d_Z+m₂d_Y+m₃d_X` for non-negative constants, a unique FGH-tripled fixed point is guaranteed to exist if `max(k₁ + l₂ + m₃, k₂ + l₁ + l₃ + m₂, k₃ + m₁) < 1`.
