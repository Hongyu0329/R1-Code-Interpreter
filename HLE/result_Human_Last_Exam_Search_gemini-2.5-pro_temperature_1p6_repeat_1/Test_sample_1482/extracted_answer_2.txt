In Interval Type-3 Fuzzy Logic Systems, the vertical cross-section for a fixed primary input `x` is an Interval Type-2 Fuzzy Set defined by an Upper Membership Function (UMF) and a Lower Membership Function (LMF) for a secondary variable `u`. Using a Gaussian paradigm, these are formulated as:

**Upper Membership Function (UMF):**
`μ̄_Ã(x)(u) = exp(-1/2 * ((u - m(x)) / σ_u)^2)`

**Lower Membership Function (LMF):**
`μ̲_Ã(x)(u) = λ * exp(-1/2 * ((u - m(x)) / σ_u*)^2)`

Where:
*   `m(x)` is the apex of the membership function for the primary variable `x`.
*   `σ_u` is the standard deviation of the upper bound.
*   `λ` (LowerScale) is a scaling factor for the lower bound (0 ≤ λ ≤ 1).
*   `σ_u*` is the standard deviation for the lower bound, typically dependent on `σ_u` and a `l` (LowerLag) parameter.
