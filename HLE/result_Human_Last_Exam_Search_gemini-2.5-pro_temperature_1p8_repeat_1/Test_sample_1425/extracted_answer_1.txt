The partition function Z for a system with Hamiltonian Ĥ = -μN̂ in the grand canonical ensemble is Z = Tr(e<sup>-β(Ĥ - μN̂)</sup>) = Tr(e<sup>2βμN̂</sup>).  Using a path integral formulation, this can be evaluated for both bosonic and fermionic systems.

The result is a product over all single-particle states *k*.

For a system of **bosons**, the partition function is:
Z<sub>bosons</sub> = &prod;<sub>k</sub> (1 - e<sup>2βμ</sup>)<sup>-1</sup>

For a system of **fermions**, the partition function is:
Z<sub>fermions</sub> = &prod;<sub>k</sub> (1 + e<sup>2βμ</sup>)
