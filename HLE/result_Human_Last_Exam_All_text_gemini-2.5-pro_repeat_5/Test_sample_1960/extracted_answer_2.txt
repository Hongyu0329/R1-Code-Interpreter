
**Function f:**
Let the notation $F^{\otimes n}$ be defined recursively as $F^{\otimes 0} = 1$ and $F^{\otimes (k+1)} = F^{\otimes k} \otimes F$.
The function $f$ maps a natural number $w$ to the formula:
$f(w) = \bot^{\otimes w}$

**Formula C:**
Using the same notation, the formula $C = C(W, m, b)$ is:
$C(W, m, b) = (\bot^{\otimes b})^{\otimes m}$
