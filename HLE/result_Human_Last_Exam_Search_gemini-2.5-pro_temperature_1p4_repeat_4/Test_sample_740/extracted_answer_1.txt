A equação não tem soluções para os valores de `λ` que são os recíprocos dos valores próprios (eigenvalues) da matriz `n x n` `M`, onde os elementos da matriz são `M_ik = 1/(n+i-k)` para `i, k = 0, 1, ..., n-1`.

Para casos específicos:
*   Se **n = 1**, o valor de `λ` para o qual não há solução é **1**.
*   Se **n = 2**, os valores de `λ` para os quais não há solução são **-6 + 4√3** e **-6 - 4√3**.
