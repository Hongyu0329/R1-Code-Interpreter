Let $q' = \lfloor \frac{n+1}{m} \rfloor$ and $r' = (n+1) \pmod m$:
- 2, if $m=n+1$.
- 3, if $m=n$.
- 4, if $\lceil \frac{n+1}{2} \rceil \le m \le n-1$.
- $2q'$, if $m < \lceil \frac{n+1}{2} \rceil$ and $r'=0$.
- $2q'+1$, if $m < \lceil \frac{n+1}{2} \rceil$ and $r'=1$.
- $2q'+2$, if $m < \lceil \frac{n+1}{2} \rceil$ and $r' \ge 2$.
