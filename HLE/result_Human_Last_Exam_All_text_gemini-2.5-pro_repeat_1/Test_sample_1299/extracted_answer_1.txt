A)[Let \(\ell = 2^a k\) with \(k\) odd. \(|U_\ell| = \tau(k)\) if \(a=0\), \(|U_\ell| = 2\tau(k)\) if \(a=1\), and \(|U_\ell| = (2+2^{a-2})\tau(k)\) if \(a \ge 2\), where \(\tau\) is the number of divisors function.] B)[Let \(\ell = 2^{e_1} p_2^{e_2} \cdots p_s^{e_s}\), where \(p_i\) are odd primes. The cardinality is \((1-d+d\prod_{i=1}^s (e_i+1))\) if \(d=1\), and for \(d=0\), it is \((2\delta_{e_1,1} + (1-\delta_{e_1,1})(2+2^{e_1-2}))\prod_{i=2}^s(e_i+1)\) where \(\delta_{i,j}\) is the Kronecker delta. Let \(C(e_1) = 1\) if \(e_1=0\), \(2\) if \(e_1=1\), \(2+2^{e_1-2}\) if \(e_1 \geq 2\). Then the expression is \(C(e_1) \prod_{i=2}^s(e_i+1)\).]
