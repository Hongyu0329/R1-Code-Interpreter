An FGH-tripled fixed point `(x, y, z)` exists and is unique if `(X,d_X), (Y,d_Y), (Z,d_Z)` are complete metric spaces and the functions `F: X*Y*Z→X`, `G: Y*X*Y→Y`, `H: Z*Y*X→Z` satisfy a set of contraction inequalities `d_X(F(x, y, z), F(u, v, w)) ≤ a1*d_X(x, u) + b1*d_Y(y, v) + c1*d_Z(z, w)`, `d_Y(G(y, x, y), G(v, u, v)) ≤ a2*d_X(x, u) + b2*d_Y(y, v)`, and `d_Z(H(z, y, x), H(w, v, u)) ≤ a3*d_X(x, u) + b3*d_Y(y, v) + c3*d_Z(z, w)`, where the non-negative constants `a_i, b_i, c_i` satisfy `a1+a2+a3 < 1`, `b1+b2+b3 < 1`, and `c1+c3 < 1`.
