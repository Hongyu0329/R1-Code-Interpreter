A unique FGH-tripled fixed point exists if (X, d_X), (Y, d_Y), (Z, d_Z) are complete metric spaces and there are non-negative constants k_f1, k_f2, k_f3, k_g1, k_g2, k_h1, k_h2, k_h3 such that d_X(F(x1,y1,z1), F(x2,y2,z2)) <= k_f1*d_X(x1,x2) + k_f2*d_Y(y1,y2) + k_f3*d_Z(z1,z2), d_Y(G(y1,x1,y1), G(y2,x2,y2)) <= k_g1*d_Y(y1,y2) + k_g2*d_X(x1,x2), d_Z(H(z1,y1,x1), H(z2,y2,x2)) <= k_h1*d_Z(z1,z2) + k_h2*d_Y(y1,y2) + k_h3*d_X(x1,x2), and k_f1+k_g2+k_h3 < 1, k_f2+k_g1+k_h2 < 1, k_f3+k_h1 < 1.
