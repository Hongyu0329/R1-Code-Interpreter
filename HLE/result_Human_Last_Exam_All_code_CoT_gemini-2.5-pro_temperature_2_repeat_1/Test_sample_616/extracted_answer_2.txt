Let $s_A = s(\det(A))$ and $s_B = s(\det(B))$. If either $A$ or $B$ is singular ($a_n=0$ or $b_n=0$), the minimum is $-\sum_{i=1}^n a_i b_i$. If both $A$ and $B$ are non-singular, the minimum is $-\sum_{i=1}^n a_i b_i$ if $s_A s_B (-1)^n=1$, and $-\sum_{i=1}^{n-1} a_i b_i + a_n b_n$ if $s_A s_B (-1)^n=-1$.
