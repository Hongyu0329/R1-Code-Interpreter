Let $V_P$ be the volume of the polytope $P$. The set of possible values for $V_P$ depends on the dimension $d$ and the volume $V$ of the simplex $T$. The result is as follows:
- For $d=1$: The set of possible values for $V_P$ is $\{V\}$.
- For $d=2$: The set of possible values for $V_P$ is $\{2V\}$.
- For $d \ge 3$: The set of possible values for $V_P$ is the interval $(2V, d!V]$. This means $2V < V_P \le d!V$.
