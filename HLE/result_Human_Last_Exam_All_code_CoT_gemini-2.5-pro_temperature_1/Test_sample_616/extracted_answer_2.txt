The minimum of the asymmetric Brockett cost function is given by $-\sum_{i=1}^n a_i b_{n-i+1}$ if $s(|A|)s(|B|) \neq -1$, and by $-\sum_{i=1}^{n-1} a_i b_{n-i+1} + a_n b_1$ if $s(|A|)s(|B|) = -1$. This can be written compactly as $-\sum_{i=1}^n a_i b_{n-i+1} + 2 a_n b_1 \cdot \mathbb{I}(s(|A|)s(|B|) = -1)$, where $\mathbb{I}(\cdot)$ is the indicator function.
