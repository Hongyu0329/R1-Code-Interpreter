Σ' = (I + A [δ]× + B [δ]×²) Σ (I + A [δ]× + B [δ]×²)^T, where A = (1-cos(||δ||))/||δ||², B = (||δ||-sin(||δ||))/||δ||³, and [δ]× is the skew-symmetric matrix of δ. Depending on the composition order convention ( `exp(δ_true) = exp(δ_new)exp(δ_est)` vs `exp(δ_true) = exp(δ_est)exp(δ_new)`), the Jacobian can also be J_r(-δ), which replaces `[δ]x` with `-[δ]x`. The code uses the latter, more common convention `J_r(-δ)`. The fundamental expression remains the covariance propagation rule `Σ' = G Σ G^T`.
