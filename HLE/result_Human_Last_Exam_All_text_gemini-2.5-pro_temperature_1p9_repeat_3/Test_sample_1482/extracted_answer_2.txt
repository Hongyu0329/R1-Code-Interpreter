The mathematical formulation for the vertical cross-section of an Interval Type-3 Membership Function is given by the Upper Membership Function (UMF) `overline{µ}_{vs}(u | x)` and Lower Membership Function (LMF) `underline{µ}_{vs}(u | x)` of the resulting Type-2 Fuzzy Set.

**UMF Formulation:**
`overline{µ}_{vs}(u | x) =`
`  { exp(-0.5 * (((u - C_U(x)) / σ_U(x))^2))`   if `µ̄_L(x) ≤ u ≤ µ̄_U(x)`
`  { exp(-0.5 * (((u - C_L(x)) / σ_L(x))^2))`   if `µ__L(x) ≤ u ≤ µ__U(x)`
`  { 0`                                           otherwise
where:
`C_U(x) = (µ̄_U(x) + µ̄_L(x)) / 2`
`C_L(x) = (µ__U(x) + µ__L(x)) / 2`
`σ_U(x) = k_U * (µ̄_U(x) - µ̄_L(x))`
`σ_L(x) = k_L * (µ__U(x) - µ__L(x))`
and `µ̄_U(x), µ̄_L(x), µ__U(x), µ__L(x)` are the four primary Gaussian bounding functions, `u` is the secondary variable, and `k_U`, `k_L` are positive constants.

**LMF Formulation:**
`underline{µ}_{vs}(u | x) = S(x) * overline{µ}_{vs}(u | x)`
where `S(x)` is a scaling function such that `0 ≤ S(x) ≤ 1`.
