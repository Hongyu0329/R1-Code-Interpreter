
**Product A** is N-(4-oxobutyl)acetamide.
**Product B** is tetrahydro-3H-pyrrolizin-3-one (also called pyrrolizidin-3-one).
**Product C** is N-acetylpyrrolidine.

Here are the structures:

**A: N-(4-oxobutyl)acetamide**
```
      O        O
      ||       //
  CH3--C--NH--CH2--CH2--CH2--C
                              \
                               H
```

**B: Tetrahydro-3H-pyrrolizin-3-one**
```
      O
     //
    /  \
   |    N----CH--CH2
   |    |         |
    \--CH2-------CH2
       |
       CH2
```
*(A more standard representation)*
```
      O=C--CH2--CH2
        |        |
        N--CH2--CH
       / \      /
      CH2-CH2
```

**C: N-acetylpyrrolidine**
```
       O
      //
     C--CH3
    /
   N----CH2
  / \   /
 CH2 CH2
  \ /
   CH2
```
